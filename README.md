# 贡献文档
所有文档在`docs`目录中，使用Markdown格式撰写，欢迎提交Pull Request。侧边栏文档目录请在`sidebars.js`中进行配置。

`main`分支更新将自动触发构建并部署。

文档编辑格式和高级用法请参考[Docusaurus文档](https://v2.docusaurus.io/docs/docs-introduction)。
